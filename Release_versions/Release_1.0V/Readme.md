# Source Code Information

## Overview

This project is the upgraded version of two previously developed utility applications:
- **Collage Maker**
- **Image Resizer**

Both utilities have been combined into a single application to provide a more streamlined and convenient user experience.

## What's New

- Unified both utilities into one application.
- Rebuilt the user interface using **Modern Tkinter (custom tkinter)**.
- Added a modern color theme for a cleaner and more polished appearance.
- Improved overall usability and navigation.
- Organized and maintainable source code structure.
- Better performance and responsiveness.

## Technologies Used

- Python
- Tkinter
- Pillow (PIL) for image processing

## Purpose

The goal of this project is to provide a lightweight, modern desktop application for common image editing tasks while maintaining a clean codebase that is easy to extend with additional utilities in the future.
